Acerca del Proyecto

Esta es una aplicación web construida con Angular 18 que simula una plataforma de información de películas y series de televisión, inspirada en The Movie Database (TMDB). El proyecto demuestra el uso de tecnologías modernas de desarrollo web y mejores prácticas de Angular.
